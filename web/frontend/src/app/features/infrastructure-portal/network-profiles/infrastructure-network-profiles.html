Transmission Conditions

Manage transmission conditions available within the platform. Only active supported transmission conditions can be used when creating experiments.

info
Network Profiles -Network profiles represent transmission conditions that are used during experiment execution.
-Only network profiles marked as supported and active can be selected by researchers.
-Supported network profiles are shown in green when active and in amber when disabled.
Only supported and active transmission conditions may be edited. Modify the Lower and Upper bound values, then click Save to apply your changes.
